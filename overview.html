LOGIC KNOWLEDGE BASE

AND Gate

The output is high (1) only if both inputs are high. Y = A • B

OR Gate

The output is high if at least one input is high. Y = A + B

NAND Gate

Universal Gate. The output is the inverse of the AND gate. Y = (A • B)'

XOR Gate

Exclusive OR. Outputs high only if inputs are different. Y = A ⊕ B
